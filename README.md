# Cut to the Chase

An AI agent that summarizes text documents using LangChain and GPT-3.5.

## Setup

1. Clone the repo
2. Install dependencies:
   pip install langchain langchain-openai langchain-core datasets python-dotenv rouge_score bert_score

3. Create a .env file in the root folder with your OpenAI key:
   OPENAI_API_KEY=your_key_here

4. Run the agent:
   python agent.py
